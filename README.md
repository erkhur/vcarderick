# VCard Coderk — React + Ant Design

Tarjeta digital de Erick Hurtado, migrada de HTML vanilla a React con Ant Design v5.

## Setup

```bash
pnpm install
pnpm run dev
```

## Imágenes requeridas

Copia tus imágenes en la carpeta `public/img/`:

| Archivo               | Descripción           |
|-----------------------|-----------------------|
| `header.png`          | Banner de cabecera    |
| `erk_avatar.jpeg`     | Foto de perfil        |
| `linkedin_premium.png`| Ícono LinkedIn        |
| `tecdata_logo.png`    | Logo Tecdata          |
| `email_icon.png`      | Ícono email           |
| `whatsapp_logo.jpg`   | Ícono WhatsApp        |

## Deploy (Vercel)

```bash
pnpm build
# Sube la carpeta dist/ a Vercel o conecta el repo
```
